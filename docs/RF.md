### Requisitos Funcionais (RF)

Sistema web single-user para geração de materiais didáticos chamado Cultura Digital alinhados à BNCC.

#### RF01 — Cadastro e gerenciamento de disciplinas
O sistema deve permitir cadastrar, editar, listar e remover disciplinas (ex.: Matemática, Ciências, História).

#### RF02 — Criação manual de unidades (aulas)
O sistema deve permitir que o usuário crie unidades (aulas) manualmente, informando o tema da aula e a série/ano.

#### RF03 — Sugestão automática de unidades via IA
O sistema deve sugerir unidades (aulas) com base na disciplina, série/ano e diretrizes da BNCC, utilizando IA.

#### RF04 — Geração automática de plano de aula
Para cada unidade, o sistema deve gerar automaticamente um plano de aula alinhado à BNCC.

#### RF05 — Geração automática de atividade avaliativa
Para cada unidade, o sistema deve gerar automaticamente uma atividade/tarefa avaliativa coerente com a unidade.

#### RF06 (Opcional) — Geração de slides de apoio
O sistema pode gerar slides de apoio para cada unidade/aula.
